New HTML DOM Events

This page listed many HTML DOM events. Create a web page named hw3-4.html that applies one or multiple events. Your code should:

  • NOT only use event(s) that we have used in past classes, such as onclick, onmouseover, onmouseout and oninput. In other words, you need to explore and learn some new events.
  • Be simple to explain the usage of the event(s) you select.
  • Be interesting. It should solve a real-world problem.
  • NOT use same example that you find from w3schools or MDN or any other websites.
  • Use comments to describe the event(s) and how your code works.
